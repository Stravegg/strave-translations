# Glossary

Terms that stay English in every locale (community-standard Siege/esports
vocabulary), plus agreed translations for common UI concepts.

## Never translate

- Strave, Rainbow Six Siege, Siege
- Map, operator, and gadget names (Bank, Clubhouse, Ash, …)
- scrim, seeding, bracket (as tournament format context allows), ban, pick,
  overtime, clutch, ace
- Hub, Ascend (Strave product names)
- Discord, Twitch, Ubisoft
